## Architecture (quick map)
- `lib/db/*`: Drizzle schema + DB client
- `lib/repos/*`: DB reads/writes (no HTTP, no UI)
- `lib/services/*`: business logic + permissions, calls repos
- `lib/actions/*`: server actions for internal UI flows (calls services)
- `lib/loaders/*`: cached read models (calls services/loaders)
- `app/api/v1/*`: external HTTP routes (calls services, never repos directly)
